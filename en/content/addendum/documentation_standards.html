Documentation Standards

1) Follow the standard way of writing documentation using Markdown, and review the existing Documents as examples.

2) Consider making your documents Multi-lingual from the very beginning.

3) Following standards for Info/Tips:

IMPORTANT: These images below do NOT have tool tip information.

4) Try to follow document naming conventions, for example: "Article - ".

5) Please note that the structure for each Book is as follow:

  • README.md is the file with Introduction
  • SUMMARY.md is the Table of Contents, which is used on the left-hand side providing you with links to the content.
